# CNN básica desde cero

Este ejemplo implementa una red neuronal convolucional (CNN) de forma
manual utilizando NumPy, con el objetivo de explicar cómo funcionan
sus componentes internos.

## Flujo del ejemplo
1. Imagen representada como matriz
2. Aplicación de un filtro (kernel)
3. Convolución 2D
4. Generación de feature maps
5. Flatten
6. Neurona final para obtener la salida

## Objetivo pedagógico
Comprender el proceso de extracción de características en una CNN sin
dependencia de frameworks de alto nivel, priorizando claridad conceptual.

Este ejemplo no incluye entrenamiento ni backpropagation.
